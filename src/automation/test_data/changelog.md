**Quant-Trad Trading Bot v1.0.1 Changelog**
### TL;DR

Updated trading bot to include maximum trades per day feature.

### What Changed

* Added `max_trades` attribute to `TradingBot` class
* Implemented `set_limit` method to set and update maximum number of trades per day

### Why This Was Done

To provide traders with more control over their daily trading activities

### Impact on the Trading Bot

This feature allows users to customize their trading experience by setting a cap on the number of trades they can execute in a single day.
